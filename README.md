# Treinos do Rafael

Aplicação front-end para controle de treinos, bioimpedância e dieta.

Principais melhorias aplicadas:
- Corrigidos IDs duplicados (`clear-diet-plan`, `diet-plan-list`) e uso de classes para ações compartilhadas.
- Adicionados atributos de acessibilidade (`aria-label`, `aria-live`, `role` onde apropriado).
- Scripts carregados com `defer` para melhorar o carregamento.
- Botões que não submetem formulários agora possuem `type="button"`.
- Adicionado `meta description` e `theme-color`.

Como rodar (localmente):
1. Abra `index.html` no navegador (é um front-end estático).
2. Se precisar integrar com backend, configure `/_sdk` ou substitua por chamadas ao seu serviço.

Contribuições são bem-vindas — abra uma issue ou PR no repositório.
